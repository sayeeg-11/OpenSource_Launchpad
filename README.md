  # Open Source Launchpad

  In collaboartion  **Social Winter of Code**

**Open Source Launchpad** is a beginner-friendly introductory workshop conducted under **Pixel Phantoms’ Month of Code (PP-MoC)**. This initiative is designed to help students understand the fundamentals of Open Source and confidently take their first steps toward real-world contributions.

## About the Program

Open Source Launchpad focuses on introducing participants to the **Open Source ecosystem**, covering essential concepts such as version control, collaborative development, and contribution workflows. The workshop emphasizes practical learning through **hands-on exposure to Git and GitHub**, helping participants understand how open-source projects are structured and maintained.

This session also serves as the **official onboarding session for PP-MoC**, where participants are introduced to the Month of Code roadmap, cohort structure, and upcoming activities.

## What You Will Learn

* Basics of **Open Source development** and community-driven projects
* **Git fundamentals** and version control concepts
* **GitHub workflows**: repositories, issues, forks, branches, and pull requests
* How to explore open-source repositories and understand contribution guidelines
* The basic process of making a **first contribution**

## Who Is This For?

* Beginners who want to start contributing to Open Source
* Students with basic programming knowledge
* Anyone curious about how real-world open-source collaboration works

## Prerequisites

* Active **GitHub account**
* Laptop/Desktop with **VS Code** installed
* Stable internet connection

## Why Open Source Launchpad?

Many beginners struggle to understand where and how to start contributing to Open Source. Open Source Launchpad aims to **remove this initial barrier** by providing clear guidance, practical exposure, and a structured introduction to contribution workflows.

By the end of the session, participants will have a clearer understanding of Open Source and the confidence to move forward with contributions as part of **Pixel Phantoms’ Month of Code (PP-MoC)**.

## About PP-MoC

**Pixel Phantoms’ Month of Code (PP-MoC)** is a structured Open Source learning and contribution initiative focused on building consistency, collaboration, and real-world development experience through guided cohorts and hands-on projects.
